# Lumina – AI Mental Health Companion

Lumina is a personalized, multimodal mental health companion web application designed to provide real-time emotional support. It leverages artificial intelligence to deliver empathetic conversations, cognitive behavioral therapy (CBT) exercises, mood tracking, and intelligent journaling.

## 🚀 Features

- 🤖 AI Chatbot with real-time empathetic responses
- 🎤 Voice emotion analysis using speech-to-text
- 😊 Facial emotion detection via webcam
- 🧠 CBT-based thought reframing system
- 📊 Mood tracking and intelligent journaling
- 🔁 Context-aware memory (remembers past conversations)
- 🔍 Retrieval-Augmented Generation (RAG) for better responses
- 🎭 Multiple persona modes (Calm Coach, Crisis Support, etc.)

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: FastAPI (Python)
- Database: MongoDB
- Authentication: JWT + SHA-256 hashing
- APIs: Groq API, Google Gemini API

## 🧠 AI Models Used

- LLaMA 3.1 (8B) – Conversational NLP
- Whisper Large v3 – Speech-to-text
- Gemini Embeddings – RAG pipeline
- Face-API.js – Facial emotion detection

## ⚙️ Key Concepts

- Multimodal Emotion Detection (Text + Voice + Face)
- Contextual Memory System
- Retrieval-Augmented Generation (RAG)
- Real-time AI inference using Groq

## 🔐 Security

- JWT-based authentication
- Password hashing using SHA-256
- Client-side facial processing (no data stored)

## 📈 Future Scope

- Wearable integration (heart rate monitoring)
- Multi-language voice support
- Offline AI processing
- Therapist dashboard integration

This project demonstrates the integration of multiple AI technologies into a single system to create a highly personalized and intelligent mental health assistant.
